# Issues

Two kinds of item:

- **`ISSUE-NNNN-*`** — a diagnosed defect or surprise, with root-cause analysis; lifecycle `draft → confirmed → resolved → reopened`.
- **`LIMITATION-NNNN-*`** — an accepted standing constraint; `current` until lifted.

This README is the intro; the index is the live list.
